# CV-builder
## Tech stack: 
### Backend:
- Flask 
- WTF-forms 
- wkhtmltopdf, pdfkit (for converting)
- html-to-json
### Frontend:
- next.js

## Features: 
- [x] choose from various CV-templates
- [x] put user's data to a template 
- [x] download in different formats(json, pdf)
